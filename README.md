# Game Space Pro

A high-performance game optimization app inspired by OPPO Game Space, built with Flutter.

## Features

- **Game Library**: Manage all your games with beautiful cover art
- **Performance Monitor**: Real-time CPU, GPU, RAM, and temperature monitoring
- **Performance Modes**: Balanced, Pro Gamer, and Low Power modes
- **Quick Boost**: One-tap system optimization
- **In-Game Toolbox**: Floating toolbox with DND, call blocking, screenshot, screen recording, voice changer, and brightness lock
- **Advanced Tools**: Crosshair overlay, FPS locker, macro keys, speed booster, RAM cleaner, and lag reducer
- **Gaming Profile**: Track your stats, achievements, and weekly play time
- **Network Optimizer**: Monitor and optimize network latency

## Getting Started

### Prerequisites

- Flutter SDK (>=3.0.0)
- Dart SDK (>=3.0.0)
- Android Studio or VS Code

### Installation

1. Clone or download this project
2. Navigate to the project directory
3. Install dependencies:

```bash
flutter pub get
```

4. Run the app:

```bash
flutter run
```

### Build APK

```bash
flutter build apk --release
```

### Build App Bundle

```bash
flutter build appbundle --release
```
